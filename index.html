LispScript Read-Eval-Print Loop

This is a toy implementation of Lisp (or Lisp-like) language in JavaScript. The following constructrs are currently supported:

  • numbers, symbols and lists
  • anonymous procedures, e.g. (lambda (x y) (+ x y))
  • variable assignment, including assigning lambdas to symbols, e.g. (define variable 1)
  • procedure calls, e.g. (procedure arg1 arg2)
  • procedures as return values

You can also take a look at the annotated source code.
